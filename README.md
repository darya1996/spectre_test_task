##Installation
1. Execute bundle

    `$ bundle install`

2. Copy .env.dist and set the environment variables (APP_ID, SECRET, PRIVATE_PEM_PATH)

    `$ cp .env.dist .env`

3. Create/migrate the db

  `$ rails db:create`

  `$ rails db:migrate`

4. `$ rails server`
