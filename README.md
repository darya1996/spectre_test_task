##Installation
1. Execute bundle

    `$ bundle install`

2. Copy application.example.yml and set the environment variables (APP_ID, SECRET)

    `$ cp /config/application.example.yml /config/application.yml`

3. Create/migrate the db

  `$ rails db:create`

  `$ rails db:migrate`

4. `$ rails server`
